## Description

Shopping cart app

## Technologies & Tools

* React
* Redux
* Redux-Saga
* Material-UI
* Webpack
* Axios

### Backend:

* Node/Express
* MongoDB/Mongoose
* Passport

### Requirements:

* Node.js installed
* MongoDB connection


### Run Client:
$ cd client  
$ npm install  
$ npm run build  
$ cd ..  
$ npm run start  
### Run Server:
$ cd server  
$ npm install  
$ node server
